# تعليمات النشر على Vercel - Afnan AI

تم إعداد المشروع ليكون جاهزاً للنشر المباشر على منصة Vercel.

## الخطوات:
1. قم برفع الملفات إلى مستودع GitHub الخاص بك.
2. اربط المستودع بمنصة [Vercel](https://vercel.com).
3. أضف المتغيرات البيئية (Environment Variables) في إعدادات المشروع على Vercel:
   - `DEEPSEEK_API_KEY`: مفتاح DeepSeek الخاص بك (تم وضع المفتاح الحالي في الكود كافتراضي، ولكن يفضل وضعه هنا للأمان).
   - مفتاح API توليد الصور (عندما يتوفر).

## هيكل المشروع:
- `index.html`: الواجهة الأمامية (Frontend).
- `api/index.js`: الخلفية البرمجية (Backend) في ملف واحد باستخدام Node.js.
- `vercel.json`: إعدادات التوجيه والنشر.

## ملاحظة حول توليد الصور:
تم تجهيز الـ Endpoint الخاص بالصور في `api/index.js` تحت مسار `/api/image`. يمكنك إضافة منطق توليد الصور هناك بمجرد حصولك على الـ API الخاص بها.
